# surface-pro-3-linux-kernel-4.2.0-43.40
Surface Pro 3 Linux Kernel 4.2.0-34.40

This patched kernel contains only the patched needed in 4.2.0
These include:
- Surface Button (windows logo on bezzel)
- Volume controls
- Power Button
- Front and rear camera
